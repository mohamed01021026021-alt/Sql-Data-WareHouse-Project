## Project Requirements
  __Building The Data Wrehouse

### Objective

Develop a modern Data Warehouse using SQL Server to integrate ERP and CRM data into a unified analytical model. The project focuses on data cleaning, transformation, and loading (ETL) to support business intelligence and reporting.

### Specifications

- Import ERP and CRM data from CSV files.
- Clean and transform the data.
- Integrate both sources into a Star Schema.
- Load the final analytical model into SQL Server.
- Create SQL queries for business reporting.
- Document the data model and ETL process.

---

## 📈 BI: Analytics & Reporting

### Objective

Develop SQL-based analytics to provide insights into:

- Customer Behavior
- Product Performance
- Sales Trends
- Business KPIs

These insights support data-driven decision-making.
